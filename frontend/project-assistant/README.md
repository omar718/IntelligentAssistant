# project-assistant README

Extension for automatically installing and launching web projects.
## Features

Task Automation: Quickly handle repetitive project setup tasks.

Smart Context: Gain insights into your project structure and dependencies.

Integrated Workflow: Seamlessly connects with your existing development tools.

## Requirements

No external dependency install is required
## Extension Settings

projectAssistant.enable: Enable/disable the assistant features.

projectAssistant.autoUpdate: Automatically check for project configuration changes.
## Known Issues

Initial indexing may take a moment on exceptionally large monorepos.
